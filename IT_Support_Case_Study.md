## Case 1: Software Crash & Data Recovery

**Client:** Small Business Owner, Abbottabad
**Issue:** Windows 10 system crash — all data inaccessible
**Tools Used:** Windows Recovery, CMD, Recuva

### Problem
Client's laptop stopped booting. All business files (Excel sheets, documents) were inaccessible. Client feared permanent data loss.

### Steps Taken
1. Booted into Windows Recovery Mode
2. Ran CHKDSK /f /r via Command Prompt
3. Used Recuva software for file recovery
4. Reinstalled Windows while preserving data partition
5. Restored all files successfully

### Result
✅ 100% data recovered
✅ System fully restored in 3 hours
✅ Client business resumed same day

## Case 2: Network Connectivity Issue

**Client:** Home User, Nawanshehr
**Issue:** WiFi connected but no internet access
**Tools Used:** CMD, Router Settings, DNS Config

### Problem
Client's laptop showed "Connected" to WiFi but no websites loaded. Issue persisted after router restart.

### Steps Taken
1. Ran `ipconfig /all` — identified IP conflict
2. Released and renewed IP: `ipconfig /release` then `ipconfig /renew`
3. Flushed DNS: `ipconfig /flushdns`
4. Changed DNS to Google (8.8.8.8 / 8.8.4.4)
5. Disabled/re-enabled network adapter

### Result
✅ Internet restored in 20 minutes
✅ DNS settings optimized for faster browsing
✅ Issue did not recur

## Case 3: Malware Removal & PC Cleanup

**Client:** Student, Abbottabad
**Issue:** PC extremely slow, suspicious popups appearing
**Tools Used:** Malwarebytes, CCleaner, Task Manager

### Problem
Student's PC was running very slow. Random popups and ads were appearing. Suspected malware infection.

### Steps Taken
1. Ran Malwarebytes scan — found 47 threats
2. Quarantined and deleted all threats
3. Used CCleaner to remove junk files (3.2GB freed)
4. Disabled startup programs via Task Manager
5. Updated Windows and antivirus definitions

### Result
✅ All malware removed
✅ PC speed improved by 60%
✅ No more popups or ads
